# Student Management System - CRUD Application

A simple PHP + MySQL based Student Management System that allows users to perform full CRUD (Create, Read, Update, Delete) operations.

---

## ✨ Features

- ➕ Add new students
- 📋 View all students in table format
- ✏️ Edit student records
- 🗑️ Delete student records
- 🎨 Clean and simple UI design

---

## 🗂️ Project Structure

```
CRUD/
├── config.php
├── index.php
├── create.php
├── update.php
├── delete.php
├── style.css
└── database.sql
```

---

## ⚙️ Requirements

- XAMPP / Laragon / WAMP
- PHP 7+
- MySQL

---

## 🚀 Installation Guide

### 1. Setup Project
- Place folder inside `htdocs/`

### 2. Create Database
- Open phpMyAdmin
- Create database: `student_crud`
- Import `database.sql`

### 3. Run Project
Open browser:
```
http://localhost/CRUD/
```

---

## 🧑‍💻 Usage

### ➕ Add Student
Click "Add Student" and fill the form.

### 📋 View Students
All students are shown in a table on homepage.

### ✏️ Edit Student
Click "Edit" button next to record.

### 🗑️ Delete Student
Click "Delete" button to remove record.
